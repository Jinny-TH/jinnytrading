# Jinny Trading v20

## 변경 내용

- 설정 > 종목 추가/수정 화면의 배당주기를 5개 항목으로 변경
  - 없음
  - 월
  - 분기
  - 반기
  - 년
- 배당주기 `없음` 선택 시 배당률 입력칸 비활성화
- 배당주기 `없음` 선택 시 DB에 `dividend_yield = null`, `dividend_cycle = '없음'` 저장
- 배당주기 월/분기/반기/년 선택 시 배당률 입력 가능
- 보유 종목 표에서 배당 없음은 `없음`, 배당 있음은 `1.15% · 월` 형식 표시
- 배당률 업데이트 버튼은 배당주기가 `없음`이 아닌 종목만 업데이트

## SQL

추가 SQL 없음.

기존 `holdings.dividend_cycle` 컬럼을 그대로 사용합니다.
