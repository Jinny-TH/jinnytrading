# Jinny Trading v27 — 개별 종목 수령 배당률 수정

## 변경 사항
- 월별 합계 배당률은 기존처럼 **선택 계좌의 총 투자금 대비**로 계산합니다.
- 개별 배당 내역은 **해당 종목 투자금(수량 × 평균단가) 대비**로 계산합니다.
- 개별 내역에 실제 월 수령률과 지급주기 기준 연환산률을 함께 표시합니다.

예시:
- 해당 종목 투자금: 10,143,750원
- 이번 수령 배당: 82,812원
- 월 수령률: 0.816%
- 월배당 종목 연환산: 9.79%

## 연환산 기준
- 월: 월 수령률 × 12
- 분기: 수령률 × 4
- 반기: 수령률 × 2
- 년: 수령률 × 1
- 없음/구분 없음: 연환산 미표시

## 동일 종목을 여러 계좌에서 보유한 경우
- `전체 계좌`에서는 현재 선택 화면에 있는 동일 종목의 투자금을 합산해 계산합니다.
- 개별 계좌 탭에서는 해당 계좌의 종목 투자금만 사용합니다.
